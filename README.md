# Simple Calculator (React.js)

A **Simple Calculator** built with **React.js** to perform basic arithmetic operations like addition, subtraction, multiplication, and division. This project has a clean and responsive UI and handles divide-by-zero errors gracefully.

## Features

- Addition, Subtraction, Multiplication, Division
- Clear input fields and result easily
- Error handling for invalid input or division by zero
- Beginner-friendly React project
